This page demonstrates the reusable web components that make up the Media Viewer application. Each component can be used independently in any web application.

Dark Mode Toggle

A toggle button for switching between light and dark themes.

<dark-mode-toggle></dark-mode-toggle>

Play/Pause Button

A button that toggles between play and pause states.

<play-pause-button></play-pause-button>

<speed-control></speed-control>

<seek-bar></seek-bar>

Fullscreen Button

A button for toggling fullscreen mode.

<fullscreen-button></fullscreen-button>

Using These Components

To use these components in your own project:

  1. Include the component scripts in your HTML file
  2. Add the component tags to your HTML
  3. Listen for events from the components
  4. Style the components using CSS variables
<!-- Include component scripts -->
<script src="components/dark-mode-toggle/dark-mode-toggle.js"></script>
<script src="components/play-pause-button/play-pause-button.js"></script>

<!-- Use the components -->
<dark-mode-toggle id="dark-mode"></dark-mode-toggle>
<play-pause-button id="play-pause"></play-pause-button>

<script>
  // Listen for events
  document.getElementById('dark-mode').addEventListener('dark-mode-change', (e) => {
    console.log('Dark mode changed:', e.detail.isDarkMode);
  });

  document.getElementById('play-pause').addEventListener('play', () => {
    console.log('Play clicked');
  });
</script>
